# 中文 Web 字體大小與顯示效果一覽表

這裡列出了常用的三大中文 Web 字體從 10px 到 100px 的大小，方便設計師觀察在什麼樣的大小下，render出來的字體較為清晰易讀。

### 例如

新細明體在 12px/13px/15px 都是可讀性不錯的點陣字形，但在 14px 就會變模糊而不好辨認。

但三大字形在 20-40px 的區段，邊緣反而因為點陣化變得很難看，直到 49px 開始開始有反鋸齒效果才比較美觀。
